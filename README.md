# Smoking Health Risk Analysis using Power BI

## Project Overview
This project analyzes healthcare data to study the impact of smoking behavior on health risks such as hypertension, cholesterol levels, and organ damage.

## Tools Used
- Power BI
- Python (Pandas) for data cleaning
- CSV datasets

## Key Insights
- Higher health risks observed among current and former smokers
- Risk increases with age and smoking duration
- Cardiovascular and lung-related risks are most prominent

## Files
- /data – Raw healthcare datasets
- /powerbi – Power BI dashboard file
